# apidoc-core
该插件用于自动扫描Java接口注释并生成接口信息对象，利用这些接口对象可以构建接口文档。

参考：https://github.com/arsframework/apidoc-maven-plugin

## 1 环境依赖
- Java JDK1.8+

## 2 部署配置
在Maven配置中添加如下依赖：
```
<dependency>
    <groupId>com.arsframework</groupId>
    <artifactId>apidoc-core</artifactId>
    <version>1.3.0</version>
</dependency>
```
